Stupid clocks

A personal project for the sake of satisfying my insanity. It is basically a collection of either stupid or insufficient ways to tell time. Two of my personal favourites are the prime clock (only displays each digit in the clock if it is prime) and the map clock (translates the current time to a position on the earth). If you have any cool ideas for a clock, feel free to contact me or submit a pull request on github.

HueIT

During my time at Chalmers University of Technology, I developed a web interface for the IT student division. The interface allows every member of the student division to access and adjust the colour and brightness of the philips hue lights placed in the division's premises. Note that the acess to the web interface requires authorisation.

Web instrument

As part of my interest in music, I wanted to develop a web interface for a digital instrument. The interface allows a user to manually set the envolope (attack, decay etc.), the overtones, some low frequency modulation of the instrument as well as what short melody it should play. It also allows a user to add more instruments in order to have them interact.

The interface relies heavily on the Web Audio API, which turns out isn't really optimised for what I wanted to do. If you play around with the interface you may notice some audio artifacts. If that is the case, try to minimise how many overtones you use.
